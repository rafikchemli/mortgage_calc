# House Affordability Calculator

A mortgage affordability dashboard for couples buying in Montreal, Quebec. Built to help answer: **"How much house can we actually afford?"**

## Features

- Dual-income support with independent housing % per person
- CMHC insurance calculation with Quebec QST
- Montreal welcome tax (droits de mutation) brackets
- Stress test at +2% rate
- Monthly cost breakdown (mortgage, taxes, maintenance, insurance, utilities)
- Upfront cash needed (down payment, welcome tax, notary, inspection)
- Dark/light mode
